## AGENT 1: DIRICHLET CHARACTERS

```
# Charon Spectral Survey — Agent 1: Dirichlet L-functions

You are running a spectral analysis experiment. This is exploration — speed over polish.

## What We Found
In elliptic curve L-functions and genus-2 curve L-functions, higher arithmetic rank
produces TIGHTER zero spacing — the opposite of what random matrix theory predicts.
The effect is consistent (every comparison negative), scales with rank (dose-response),
and survives conductor controls. We need to know if this is universal.

## Your Assignment: Dirichlet Characters
Dirichlet L-functions are the simplest non-trivial L-functions. If the phenomenon
appears here, it's as universal as possible.

## Step 1: Get the data
Use the LMFDB API to download Dirichlet character L-function zeros.
- API: https://www.lmfdb.org/api/lfunc/lfunctions/?degree=1&limit=200
- Or scrape from: https://www.lmfdb.org/L/degree1/
- You need: the first 20+ zeros (imaginary parts on critical line) for each L-function
- Group by ORDER of the character (order 1, 2, 3, 4, 5+)
- Also record the conductor (modulus) for each

If the API is difficult, try downloading from:
https://www.lmfdb.org/api/lfunc/lfunctions/?degree=1&order=2&limit=50
Vary the order parameter to get different structural levels.

Alternative approach if API is slow: LMFDB has a download feature. Go to
https://www.lmfdb.org/L/degree1/ and look for bulk download or API documentation.

Target: at least 50 objects per order level, ideally 200+.

## Step 2: Compute normalized gaps
For each L-function with 20+ zeros:
- Compute gaps: gap_i = zero_{i+1} - zero_i for i = 1..19
- Normalize each gap by the mean gap for that object (so gaps are scale-free)
- Store as a 19-dimensional vector per object

## Step 3: Compare by order
- Reference group: order-1 characters (trivial/quadratic)
- Comparison groups: order 2, 3, 4, 5+
- For each gap position (1-19), compute Cohen's d between reference and comparison
- Cohen's d = (mean_ref - mean_comp) / pooled_std
- RECORD THE SIGN. Negative = tighter (our phenomenon). Positive = wider (RMT prediction).

## Step 4: Dose-response
- If you have 3+ order levels with sufficient data, check:
  does the effect get stronger with higher order?
- Compute Spearman correlation between order and mean Cohen's d

## Step 5: Conductor control
- Split objects into 3 bins by conductor (low/medium/high)
- Within each bin, repeat the order comparison
- Does the sign inversion persist in all bins?

## Output
Save results to: F:\Prometheus\charon\spectral_survey\results\dirichlet.json

Format:
{
  "family": "dirichlet",
  "structural_parameter": "character_order",
  "levels": [...],
  "n_per_level": [...],
  "cohens_d_matrix": {"order_2": [...19 values...], "order_3": [...], ...},
  "fraction_negative": 0.XX,
  "dose_response_spearman": X.XX,
  "conductor_independent": true/false,
  "verdict": "SIGN_INVERSION_CONFIRMED/PARTIAL/NO_EFFECT/OPPOSITE/INSUFFICIENT_DATA"
}

Verdict thresholds (LOCKED BEFORE SEEING DATA):
- CONFIRMED: ≥85% of gap comparisons negative
- PARTIAL: 65-85% negative
- NO_EFFECT: 35-65%
- OPPOSITE: ≤35%
- INSUFFICIENT_DATA: fewer than 30 objects in any comparison group

## Important
- Work fast. Cache LMFDB responses to disk so you don't re-download.
- If the API is slow or rate-limited, work with whatever you get first, note the N.
- Do NOT try to explain the results. Just measure and report.
- Do NOT strip mechanisms. Just get the sign, magnitude, and dose-response.
- Print progress as you go so I can monitor.
```

# Charon Spectral Survey — Agent 2: Number Field L-functions (Dedekind Zeta)

You are running a spectral analysis experiment. This is exploration — speed over polish.

## What We Found
In elliptic curve and genus-2 L-functions, higher arithmetic rank produces TIGHTER
zero spacing — opposite to random matrix theory predictions. Consistent sign, dose-
response scaling, conductor-independent. We're testing universality.

## Your Assignment: Number Fields (Dedekind Zeta Functions)
Every number field has a Dedekind zeta function. The structural parameter is the
DEGREE of the number field (linear, quadratic, cubic, quartic, quintic, sextic).

## Step 1: Get the data
LMFDB has ~500,000 number fields. We need their Dedekind zeta zeros.

Approach A — via L-function database:
- https://www.lmfdb.org/api/lfunc/lfunctions/?degree=2&limit=200 (quadratic fields)
- Vary degree for higher fields
- Need first 20+ zeros per L-function

Approach B — via number field database then link to L-functions:
- https://www.lmfdb.org/api/nf/fields/?degree=2&limit=200
- Each number field has a linked Dedekind zeta function
- Follow the link to get zeros

Group by DEGREE: 2 (quadratic), 3 (cubic), 4 (quartic), 5 (quintic), 6 (sextic).
Degree-1 is just Riemann zeta — only one object, can't be a group.

Also record: discriminant (absolute value) as the conductor analog.

Target: at least 50 objects per degree, ideally 200+.

## Step 2: Compute normalized gaps
For each Dedekind zeta with 20+ zeros:
- gaps: gap_i = zero_{i+1} - zero_i for i = 1..19
- Normalize by mean gap
- 19-dimensional vector per object

## Step 3: Compare by degree
- Reference group: degree-2 (quadratic fields — largest, best-studied)
- Comparison: degree 3, 4, 5, 6
- Cohen's d per gap position. RECORD THE SIGN.
- Negative = tighter gaps with higher degree = our phenomenon

## Step 4: Dose-response
- Spearman correlation: degree vs mean Cohen's d across gaps
- Monotonic negative correlation = dose-response confirmed

## Step 5: Discriminant control
- Split into 3 bins by |discriminant|
- Repeat degree comparison within each bin
- Persistence across bins = discriminant-independent

## Output
Save to: F:\Prometheus\charon\spectral_survey\results\number_fields.json

Same JSON format as protocol. Include:
- family, structural_parameter, levels, n_per_level
- cohens_d_matrix, fraction_negative
- dose_response_spearman
- conductor_independent (here: discriminant_independent)
- verdict

Verdict thresholds LOCKED: ≥85% negative = CONFIRMED, 65-85% = PARTIAL,
35-65% = NO_EFFECT, ≤35% = OPPOSITE.

## Important
- Dedekind zeta zeros may not be precomputed for all fields in LMFDB.
  If zeros aren't available via API, check if LMFDB stores them elsewhere.
- If only some degrees have zeros available, analyze what you can and
  note which degrees had insufficient data.
- Cache everything. Work fast. Report what you find.

## AGENT 4: MAASS FORMS

```
# Charon Spectral Survey — Agent 4: Maass Form L-functions

You are running a spectral analysis experiment. Speed over polish.

## What We Found
Higher arithmetic structure → tighter zero spacing. Confirmed in 3 families.
Testing universality across all L-function types.

## Your Assignment: Maass Forms
Maass forms are NON-HOLOMORPHIC automorphic forms. They have a continuous
spectral parameter R (the eigenvalue of the hyperbolic Laplacian) instead
of a discrete weight like classical modular forms.

This is a critical test: if the phenomenon appears for Maass forms, it
extends beyond algebraic/holomorphic objects into the analytic world.

## Step 1: Get the data
- LMFDB Maass forms: https://www.lmfdb.org/api/mf/maass/
- Or browse: https://www.lmfdb.org/ModularForm/GL2/Q/Maass/
- Each Maass form has an associated L-function with zeros
- Need: first 20+ zeros, spectral parameter R, level (conductor analog)

The structural parameter here is different from other families. R is
CONTINUOUS, not discrete. So instead of grouping by rank=0,1,2,3 you'll
need to bin R into ranges.

Approach: 
- Download Maass forms with their R values and levels
- Bin R into quartiles or quintiles
- Compare gap statistics across R bins

Alternative structural parameter: the LEVEL of the Maass form (analogous
to conductor). Group by level and check if higher level → tighter gaps.
Try BOTH R and level as structural parameters.

Target: at least 50 objects per bin.

## Step 2: Compute normalized gaps
Same protocol: gap_i = zero_{i+1} - zero_i, normalize by mean gap.

## Step 3: Compare by R (and by level)
- Reference group: lowest R bin (or lowest level bin)
- Compare to higher bins
- Cohen's d per gap. SIGN matters.

## Step 4: Dose-response
- Spearman: R (or level) vs mean Cohen's d
- Check both structural parameters independently

## Step 5: Level control (if using R as primary parameter)
- Bin by level, repeat R analysis within bins
- Or vice versa: bin by R, repeat level analysis within bins

## Output
Save to: F:\Prometheus\charon\spectral_survey\results\maass.json

Include BOTH analyses if you run both R and level as structural parameters:
{
  "family": "maass",
  "analysis_by_R": { ...standard format... },
  "analysis_by_level": { ...standard format... },
  "verdict_R": "...",
  "verdict_level": "..."
}

Verdict thresholds LOCKED: ≥85% = CONFIRMED, 65-85% = PARTIAL,
35-65% = NO_EFFECT, ≤35% = OPPOSITE.

## Important
- Maass forms are the most different family from our confirmed cases.
  EC and genus-2 are algebraic. Modular forms are holomorphic. Maass
  forms are neither. A positive result here would be the strongest
  evidence for universality.
- A NEGATIVE result is also valuable — it would mean the phenomenon
  requires algebraic structure specifically.
- R is continuous, so your binning choices affect the result. Try
  quartiles first, then quintiles, and check if the sign is robust
  to different bin boundaries.
- Work fast. Cache. Report.
```
