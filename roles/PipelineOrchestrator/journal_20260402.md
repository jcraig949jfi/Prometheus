Directory structure created: `forge/v2/` and `forge/v3/` with full agent subdirectories.

### Architectural decisions

1. Each tier's output is the next tier's primitives
2. Tiers run in parallel — lower tiers never stop (substrate generators)
3. Nous at each tier mines DIFFERENT things (triples vs combinations vs substrate pairs)
4. LLM randomness is the mutation operator at every tier
5. Each tier has its own battery, ledger, forge directory, and Nemesis
