### P1.1 Mahler Measure Bridge Test (EXECUTED)

**Hypothesis**: Knot Alexander polynomial Mahler measures match EC L-values at 10^-6 precision (Aporia prediction: >= 50 matches).

**Method**:
1. Computed Mahler measures for all 2,977 knots with Alexander polynomial data
2. Two methods: root-finding (exact) and numerical integration (10K-point quadrature)
3. Root-finding is authoritative; numerical has ~0.0002 error for degree-10 polynomials
4. Matched log(M) values against 200K EC regulators from Postgres

**Results**:
- 2,961 non-trivial Mahler measures (16 trivial M=1)
- 43 knots below Lehmer's constant 1.17628... (interesting for MATH-0042)
- Range: [1.0, 30.4], mean: 7.45, median: 6.75

**Bridge test (log(M_knot) vs EC regulator)**:

| Tolerance | Real matches | Null matches | Enrichment |
|-----------|-------------|-------------|------------|
| 10^-6 | 81 | 100 | 0.81x |
| 10^-5 | 1,213 | 930 | 1.30x |
| 10^-4 | 9,729 | 9,202 | 1.06x |

**Verdict: CONSISTENT WITH NULL.** No enrichment above random expectation.

**z=10.16 artifact (full null test, 1.85M regulators, 45 min):**
The full null test appeared to show z=10.16 enrichment, but this used a UNIFORM null — drawing random values uniformly from [min, max] of log(M). Real Mahler values cluster heavily at log(2), log(3), log(4), log(8) etc. (112 knots at log(4) alone). These pile-up points coincide with dense regions of the regulator distribution because small integers are universally common. A permutation null (shuffling Mahler assignments, preserving distribution) gives identical count. **This is the prime atmosphere again** — shared arithmetic structure, not cross-domain signal.

**Definitive L-value test (after index creation):**
Queried 500K rank-0 EC L(E,1) values from lfunc_lfunctions (1.74M EC L-funcs total). Matched against log Mahler measures with PERMUTATION null (shuffle which knot has which Mahler measure):

| Tolerance | Real matches | Permutation null | z-score |
|-----------|-------------|-----------------|---------|
| 10^-6 | 839 | 839 +/- 0 | 0.00 |
| 10^-5 | 6,675 | 6,675 +/- 0 | 0.00 |
| 10^-4 | 69,100 | 69,100 +/- 0 | 0.00 |

z = 0.00 exactly. The match count depends only on the set of values, not on which knot has which measure. Shuffling assignments doesn't change anything.

**Caveats** (why this is not a full kill of Boyd's conjecture):
1. Boyd's conjecture relates Mahler measures of **A-polynomials** (2-variable), not Alexander polynomials (1-variable), to L-values
2. The connection goes through specific knots (e.g., 2-bridge knots, torus knots) — a blanket test of all 2977 knots dilutes signal
3. We tested L(E,1), but Boyd relates to L(E,2) = L'(E,0)/(2*pi)

**Classification: Alexander-to-L-value bridge KILLED. Boyd's conjecture (A-polynomial) remains OPEN — needs different data.**
