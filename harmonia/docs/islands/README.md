# The Islands — Domains That Speak Unnamed Phonemes

Five mathematical domains that decouple from the main tensor network when measured through the 5 known phonemes (complexity, rank, symmetry, arithmetic, spectral), yet show rank 3-4 coupling via the cosine scorer. The gap between what cosine finds and what phoneme finds is the signal — each island broadcasts on a frequency the phoneme alphabet hasn't named yet.

Each island is named for a Greek figure whose myth mirrors its mathematical character.

The islands are not noise. They are the frontier — where the phoneme alphabet needs to grow.
