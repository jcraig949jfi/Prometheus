# The Maths — Diverse Mathematical Implementations for Noesis

Each subdirectory contains Python implementations of mathematical operations
from a specific field. Every function must be:
- Pure Python + numpy (no exotic dependencies)
- Callable with standard inputs (scalar, array, matrix)
- Typed: documented input_type and output_type
- Tested: runs without crashing on at least 3 test inputs

These become organisms in the Noesis tensor. The value is in the CROSS-FIELD
compositions — what happens when you pipe tropical geometry into p-adic numbers
into a Ramsey theory bound.
