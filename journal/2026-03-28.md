**Tensor vs Random:** Random chains outperformed tensor-guided chains (12% vs 4% execution rate). Root cause: tensor scores conceptual affinity but has no type compatibility awareness. The tensor recommends pairs that can't operationally connect. Fix identified: pre-filter by type compatibility using `compatible_chains()`, then weight by tensor score.
