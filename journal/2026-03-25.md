## Key Decisions

1. **Coeus sits between Nous and Hephaestus** — not inside either. Clean separation of
   concerns. It's the intelligence layer, not a scorer.

2. **Implementability is the only score that predicts forge success.** Reasoning,
   metacognition, hypothesis generation all weight 0.000. Rewrote Nous prompt accordingly.

3. **Code-first extraction** — check for code before checking for decline signals. Fixed
   the 30% false-scrap rate.

4. **NCD as quality floor, not a tool** — every forged tool must beat compression distance.
   This raises the bar from "better than random" to "better than information theory."

5. **Adversarial data never enters training** — Batch 3 proved mixing costs 25 points on
   metacognition. Enforced in code with provenance gate.

6. **MAP-Elites over random bag** — quality-diversity ensures 100 diverse adversarial tasks
   covering the full failure boundary, not 100 variations on the same trick.
