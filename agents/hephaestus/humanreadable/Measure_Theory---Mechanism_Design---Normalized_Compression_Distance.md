1. **Parse** each prompt and candidate answer into a finite set \(P=\{p_1,\dots,p_n\}\) of atomic propositions. Each proposition is a struct with fields:

   - `value`: for numeric propositions the parsed float; otherwise `None`.  
   Parsing uses deterministic regexes for patterns like `\bif\b.*\bthen\b`, `\b(>|<|>=|<=)\b`, `\bnot\b`, `\band\b`, `\bor\b`, and extracts numbers with `[-+]?\d*\.?\d+`. The result is stored as a NumPy structured array `props` of length \(n\).  

2. **Compute a measure** on the σ‑algebra generated by \(P\). For each proposition \(p_i\) compute its Normalized Compression Distance (NCD) to the reference answer \(r\):  
   \[
   \text{NCD}_i = \frac{C(p_i\!\Vert\!r)-\min\{C(p_i),C(r)\}}{\max\{C(p_i),C(r)\}},
   \]  
   where \(C(x)=\text{len}(\text{zlib.compress}(x.encode()))\). Convert to a weight \(w_i=\exp(-\text{NCD}_i)\) and store in a NumPy array `w`. The measure of any subset \(A\subseteq P\) is \(\mu(A)=\sum_{i\in A} w_i\).  

3. **Score** a candidate answer \(a\) using a VCG‑style proper scoring rule. Let \(o_i\in\{0,1\}\) be the truth value of \(p_i\) in the reference answer (computed by evaluating the parsed propositions). Let \(p_i^{(a)}\in\{0,1\}\) be the truth value assigned by the candidate (same evaluation). The weighted Brier loss is  
   \[
   L(a)=\sum_{i=1}^{n} w_i\bigl(p_i^{(a)}-o_i\bigr)^2 .
   \]  
   The mechanism‑design payment (the score we return) is the negative loss:  
   \[
   S(a)=-L(a).
   \]  
   Higher \(S\) indicates better alignment with the reference under the compression‑based measure. All operations are vectorized with NumPy; no external models or APIs are used.
