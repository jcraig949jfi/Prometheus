     * Conditionals (`if … then …`) → antecedent \(a\), consequent \(c\).  
     * Causal verbs (`causes`, `leads to`, `results in`) → edge \(a→c\).  
     * Numeric values → float constants.  
   - Build a proposition‑graph \(G=(V,E)\) where each vertex is a literal (possibly negated) and edges encode logical relations (implication, equivalence, ordering).  
   - Assign each vertex a base weight \(w_i=1\).  

2. **Measure Construction**  
   - Define a σ‑algebra \(\mathcal{F}\) on the power set of \(V\) generated by the edge‑sets (each edge corresponds to a measurable event “the relation holds”).  
   - Define a Lebesgue‑like measure \(\mu\) on \(\mathcal{F}\) by \(\mu(S)=\sum_{v\in S} w_v\).  
   - For each candidate answer \(A\), compute the set \(S_A\subseteq V\) of literals that are satisfied (truth‑evaluated via a simple deterministic evaluator handling ¬, ∧, ∨, →, and numeric comparisons).  
   - The raw measure score is \(m(A)=\mu(S_A)\).  

3. **Nash Equilibrium Layer**  
   - Treat each candidate answer as a pure strategy in a normal‑form game where the payoff to strategy \(A\) is \(u_A = m(A) - \lambda \cdot \text{penalty}(A)\).  
   - Penalty encodes violation of hard constraints (e.g., a conditional whose antecedent is true but consequent false).  
   - Compute mixed‑strategy Nash equilibria via solving the linear complementarity problem (LCP) using `numpy.linalg.lstsq` on the payoff matrix \(U_{ij}=u_i\) (zero‑sum simplification: opponent pays \(-u_i\)).  
   - The equilibrium probability \(p_i^*\) gives the normalized weight of answer \(i\).  

4. **Sensitivity Analysis**  
   - Perturb each input token by synonym substitution or negation flip (generated via a small predefined lexicon).

   - Sensitivity score for answer \(A\) is \(s(A)=\frac{1}{K}\sum_{\delta}|p_\delta^*(A)-p^*(A)|\).  
   - Final MWNSS score: \(\text{score}(A)=p^*(A) \cdot \exp(-\beta\, s(A))\) with \(\beta=0.5\).
