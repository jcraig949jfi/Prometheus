## Nous Analysis

**Algorithm**  
We treat each candidate answer as a node in a directed graph \(G=(V,E)\) whose vertices are propositions extracted from the answer text.  
1. **Structural parsing** – Using only regex and the stdlib we pull out:  
   - atomic predicates (e.g., `X > Y`, `not P`, `if A then B`)  
   - binary relations (comparatives, ordering, causal arrows, spatial prepositions)  
   - numeric constants.  
   Each predicate becomes a feature vector \(f_i\in\mathbb{R}^d\) where dimensions encode: polarity (±1 for negation), relation type (one‑hot), grounded affordance (see below), and normalized numeric value.  
2. **Gauge‑theoretic connection** – We define a connection \(A_{ij}\) on each edge \(e_{ij}\) as the difference of the affine frames of the source and target nodes:  
   \[
   A_{ij}=f_j - f_i .
   \]  
   Parallel transport of a vector \(v\) from node \(i\) to \(j\) is \(v' = v + A_{ij}\).  
   The gauge‑invariant similarity between two answers \(a\) and \(b\) is the dot product after transporting all vectors of \(a\) to a common reference frame (chosen as the node with highest degree):  
   \[
   s_{\text{gauge}}(a,b)=\sum_{i\in V_a} \bigl(f_i^{\parallel}\bigr)\cdot\bigl(g_i^{\parallel}\bigr),
   \]  
   where \(\parallel\) denotes transport to the reference frame. This implements a local invariance: re‑labeling of internal variables (a gauge transformation) leaves the score unchanged.  
3. **Embodied grounding** – Affordance vectors are built from a small lookup table (e.g., `push` → [0,1,0], `above` → [0,0,1], `weight` → [norm,0,0]) and concatenated to the predicate feature. Thus symbols are tied to sensorimotor dimensions.  
4. **Reinforcement‑learning scoring** – Each candidate answer starts with a Q‑value \(Q_a=0\). After computing \(s_{\text{gauge}}(a, r)\) where \(r\) is a reference answer generated by a rule‑based solver, we define a reward  
   \[
   R_a = \operatorname{clip}\bigl(s_{\text{gauge}}(a,r),0,1\bigr).
   \]  
   We update via a temporal‑difference step:  
   \[
   Q_a \leftarrow Q_a + \alpha\bigl(R_a + \gamma\max_{a'}Q_{a'} - Q_a\bigr),
   \]  
   with fixed \(\alpha=0.2,\;\gamma=0.9\). The final score is \(Q_a\).
