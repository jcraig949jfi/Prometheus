*Operations* (per iteration t)  
1. **Exploration‑exploitation selection** – compute UCB score for each arm:  
   `UCB_i = (rewards[i]/pulls[i]) + α * sqrt(log(t+1)/pulls[i])` (with `α=1.0`).  
   Choose arm `i* = argmax UCB_i`. If `pulls[i*]==0` force exploration.  
2. **Swarm perturbation** – generate `m` mutant answers by applying stochastic edit operators on `G_{i*}`:  
   - node flip (negation insertion/deletion)  
   - edge weight perturbation (±ε)  
   - edge rewiring (randomly reconnect two nodes preserving type).  
   Each mutant yields a hypergraph `G'`.  
3. **Falsification test** – for each `G'` run a lightweight constraint propagator:  
   - propagate transitivity over ordering edges (`A<B ∧ B<C ⇒ A<C`).  
   - apply modus ponens on conditional edges (`if A then B` ∧ `A` ⇒ `B`).  
   - detect contradictions (a node and its negation both true).  
   If a contradiction is found, record the mutant as a counter‑example and push it onto `F`.  
4. **Reward assignment** – base reward `r = 1 - (|F|/k)` (fraction of buffer free of contradictions).  
   Update `rewards[i*] += r`, `pulls[i*] += 1`.  
5. **Iterate** until a budget of `T` pulls is exhausted; final score for each candidate is its average reward `rewards[i]/pulls[i]`.
