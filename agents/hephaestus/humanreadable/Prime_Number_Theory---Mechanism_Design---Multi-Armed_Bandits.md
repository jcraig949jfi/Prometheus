## Nous Analysis

**Algorithm**  
1. **Proposition extraction** – For each candidate answer, apply a handful of regex patterns to pull atomic clauses:  
   *Negation* (`\bnot\b|\bno\b`), *comparative* (`\b(greater|less|more|fewer)\b.*\bthan\b`), *conditional* (`\bif\b.*\bthen\b`), *causal* (`\bbecause\b|\bleads to\b|\bresults in\b`), *numeric* (`\b\d+(\.\d+)?\b`), *ordering* (`\b(first|second|last|before|after)\b`).  
   Each matched clause is stripped of punctuation and lower‑cased to form a proposition string \(p_i\).

2. **Prime encoding** – Maintain a growing list of the first \(K\) primes generated by a simple sieve (numpy `np.arange` + boolean mask).  
   A dictionary `prop2prime` maps each unique proposition to the next unused prime; the associated weight is `w_i = log(prime_i)`.  
   For an answer \(A\) with propositions \(\{p_{i}\}\), compute its **informativeness**  
   \[
   I_A = \sum_{i} w_i \;=\; \sum_{i} \log(\text{prime}(p_i))
   \]
   (implemented as a dot product of a binary indicator vector with a numpy array of log‑primes).

3. **Mechanism‑design scoring rule** – Treat each answer as a report from a self‑interested agent.  
   Use a proper scoring rule: the expected reward for reporting confidence \(c\) is  
   \[
   R(c, \text{correct}) = c\cdot\log(\text{correct}) + (1-c)\cdot\log(1-\text{correct}),
   \]
   where “correct” is 1 if the answer matches a hidden ground‑truth label (available only for a small validation set) and 0 otherwise.  
   To avoid needing the label at test time, we maintain a Beta prior \(\text{Beta}(\alpha_A,\beta_A)\) on the latent correctness probability. The posterior mean \(\hat{p}_A = \alpha_A/(\alpha_A+\beta_A)\) serves as the estimated confidence.

4. **Multi‑armed bandit update** – Each answer is an arm. After each validation round, observe binary outcome \(o\in\{0,1\}\) and update:  
   \[
   \alpha_A \leftarrow \alpha_A + o,\quad \beta_A \leftarrow \beta_A + (1-o).
   \]  
   For ranking, draw a Thompson sample \(\theta_A\sim\text{Beta}(\alpha_A,\beta_A)\) and compute the final score  
   \[
   S_A = \theta_A \cdot I_A .
   \]  
   The answer with highest \(S_A\) is selected. All operations use only numpy arrays and Python’s `random.betavariate` (or numpy’s `np.random.beta`).
