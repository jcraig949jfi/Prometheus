1. **Parsing & typing**

   - Each extracted proposition pᵢ is assigned a **type** from a fixed finite set (e.g., `Entity`, `Quantity`, `Relation`, `Event`). Types are stored as strings; the pair (pᵢ, typeᵢ) forms a **typed term**.  

2. **Prime encoding**  
   - Maintain a growing list of primes generated by a simple sieve (numpy `uint64`).  
   - When a new typed term appears, assign it the next unused prime prᵢ.  
   - A set of terms is represented by the **product** of their primes. Conjunction corresponds to multiplication; testing whether a set S entails a term t reduces to checking divisibility: `product(S) % pr_t == 0`.  
   - Negation is handled by storing a separate “negative prime” ¬prᵢ (the same prime with a flag) and treating a conjunction that contains both a term and its negation as inconsistent (product contains prᵢ², which we reject).  

3. **Adaptive control of inference rules**  
   - Define a small fixed rule base (modus ponens, transitivity, contrapositive, arithmetic ordering). Each rule rⱼ has an associated weight wⱼ stored in a numpy array.  
   - For a given candidate, forward‑chain using the current weights: each time a rule fires, add its weight to a **confidence score** c.  
   - Compare c to a binary target y (1 if the candidate matches a human‑provided gold answer, else 0) using a simple squared error e = (c − y)².  
   - Update weights with a self‑tuning regulator‑style update:  
     `w ← w − η · ∂e/∂w` where ∂e/∂w = 2·(c − y)·fⱼ and fⱼ is the firing count of rule rⱼ in this forward pass. η is a small constant (e.g., 0.01). This is pure numpy arithmetic, no external model.  

4. **Scoring logic**  
   - After a fixed number of online updates (e.g., 5 passes over the training pairs), the final weight vector encodes how predictive each rule is for the task.  
   - To score a new candidate, compute its confidence c using the adapted weights and then normalize:  
     `score = sigmoid(c) = 1 / (1 + exp(−c))`.  
   - The score lies in [0,1]; higher values indicate greater logical consistency and alignment with the gold answer under the learned rule weighting.
