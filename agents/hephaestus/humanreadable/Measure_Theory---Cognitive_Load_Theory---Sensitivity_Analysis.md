**Algorithm**  
1. **Parsing stage** – Using a fixed set of regex patterns, extract from both the prompt (reference) and each candidate answer a list of atomic propositions \(p_i\). Each proposition carries a type tag \(t_i\in\{\text{negation},\text{comparative},\text{conditional},\text{numeric},\text{causal},\text{ordering}\}\), a Boolean truth value \(v_i\) (determined by evaluating the extracted clause against a simple world model built from the prompt), and a raw token span \(s_i\). The collection \(\{p_i\}\) forms the atoms of a σ‑algebra \(\mathcal{F}\) generated by all possible truth assignments.  
2. **Measure construction** – Define a weight \(w_i\) for each proposition as  
   \[
   w_i = \underbrace{\alpha\frac{1}{|P|}}_{\text{intrinsic load}} \;+\; \underbrace{\beta\frac{|S\setminus\bigcup_i s_i|}{|S|}}_{\text{extraneous load}} \;+\; \underbrace{\gamma\frac{|\{p_j:v_j=\hat v_j\}|}{|P|}}_{\text{germane load}},
   \]  
   where \(P\) is the set of propositions in the answer, \(S\) the set of all tokens in the answer, \(\hat v_j\) the reference truth value from the prompt, and \(\alpha,\beta,\gamma\) are fixed scalars (e.g., 0.4,0.3,0.3). The vector \(\mathbf{w}\in\mathbb{R}^{|P|}\) defines a discrete measure \(\mu\) on \((\{p_i\},\mathcal{F})\) by \(\mu(A)=\sum_{p_i\in A}w_i\).  
3. **Sensitivity analysis** – Compute a baseline score \(S_0=\sum_i w_i v_i\). For each proposition \(i\), flip its truth value (\(\tilde v_i=1-v_i\)) while keeping others fixed, yielding a perturbed score \(S_i\). The sensitivity (influence) is \(\delta_i=|S_i-S_0|\). Collect \(\boldsymbol{\delta}\) as a NumPy array.  
4. **Final scoring** – Normalize sensitivities \(\tilde\delta_i=\delta_i/(\max\boldsymbol{\delta}+ \epsilon)\). The answer score is  
   \[
   \text{Score}= \frac{\sum_i w_i v_i (1-\tilde\delta_i)}{\sum_i w_i},
   \]  
   i.e., a weighted average of truth‑value contributions penalized by propositional sensitivity. All operations use NumPy arrays; no external models are invoked.
