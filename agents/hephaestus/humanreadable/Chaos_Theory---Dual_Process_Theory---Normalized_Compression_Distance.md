2. **System 1 similarity** – Compute the Normalized Compression Distance between *A* and a reference answer *R* (generated by applying the same extraction rules to *P* and filling in expected values via simple arithmetic). Using only `zlib` from the standard library:  
   `NCD(A,R) = (|zlib.compress(A+R)| - min(|zlib.compress(A)|,|zlib.compress(R)|)) / max(|zlib.compress(A)|,|zlib.compress(R)|)`.  
   System 1 score `S1 = 1 - NCD`.

Let `V` be the total number of violations; `System 2 score S2 = 1 - V / (|C| + |E| + 1)`.  

4. **Chaos‑sensitivity term** – Generate *k* (e.g., 5) perturbed copies of `G` by randomly swapping two independent clauses or flipping a negation. For each copy recompute `S2`.
