**Harmonia's peak state was not intelligence — it was grounding.**

### The Plan

**Phase 1: Triage** — Classify 490 math problems into three buckets:

**Phase 2: Bridge** — Connect Aporia to Harmonia's engine:

- Import from `cartography/shared/scripts/` (falsification_battery, known_truth_battery)

**Phase 3: Instrument** — Run the Harmonia pattern on Bucket A problems:
- State prediction precisely
- Pull data

- Document result (both illuminated and shadow)

**Phase 4: Blind Trials** — James's "open book test" idea:
- Take problems the LLM classified as `solved`
- Point the instrument at the underlying data WITHOUT telling it the answer
- Can it independently recover the known result?
- This validates both the instrument AND the AI's ability to use it

### Key Insight: The Open Book Test

James posed a genuinely interesting question: "Can AI pass an open book test when the answer is right there in the weights?"

This is deeper than it sounds. The LLM *knows* that Connes embedding was disproved (it said so with confidence 10/10). But can it *compute* why? Can it point the instrument at quantum correlation data and measure the C_co != C_ts separation?

If yes: the AI isn't just reciting — it's connecting knowledge to measurement. That's the Harmonia state.
If no: the AI is a parrot with good memory. Still useful (status filter), but not an instrument.

The blind trial protocol tests this directly. It's the difference between knowing that BSD holds and being able to show that rank = analytic_rank across 3.8M curves.
