# Cluster Algebra Positivity — Deep Research for Ergon

## What Are Cluster Algebras?

A **cluster algebra** (Fomin-Zelevinsky 2002) is a commutative ring generated by **cluster variables** produced through **mutation**. A **seed** is a pair (x, B): a cluster x = (x_1, ..., x_n) and a skew-symmetrizable integer matrix B (encoding a weighted quiver). Mutation at index k replaces x_k with x_k' via the **exchange relation**: x_k * x_k' = (product over positive B-entries) + (product over negative B-entries), while B mutates by a matrix rule. The cluster algebra is the subring of Q(x_1, ..., x_n) generated by all reachable cluster variables. The exchange graph can be finite (classified by Dynkin diagrams) or infinite.

## The Laurent Phenomenon and Why Positivity Is Surprising

**Laurent phenomenon** (Fomin-Zelevinsky 2002): every cluster variable, when expressed in terms of any fixed initial cluster, is a Laurent polynomial — a polynomial divided by a monomial in the initial variables. This is already remarkable: the exchange relation involves division, yet denominators always cancel to monomials. The **positivity conjecture** strengthens this: all coefficients of these Laurent polynomials are nonnegative integers. This is surprising because mutation sequences can be arbitrarily long and involve repeated cancellations. That positivity survives this combinatorial explosion is deeply non-obvious.

## What Has Been Proved

**Skew-symmetric case (Lee-Schiffler 2015, Annals of Mathematics):** The positivity conjecture holds for all skew-symmetric cluster algebras (B is skew-symmetric, equivalently the quiver has no 2-cycles). This was the main breakthrough, dedicated to Zelevinsky. Their proof uses an inductive formula for cluster variables via T-paths on triangulated surfaces, generalized to arbitrary quivers.

**Skew-symmetrizable case (Gross-Hacking-Keel-Kontsevich 2018):** Using **scattering diagrams** from mirror symmetry, GHKK proved positivity for all skew-symmetrizable cluster algebras. Their construction: associate to each seed a scattering diagram (a collection of walls in R^n carrying formal power series), then define **theta functions** via broken lines — piecewise-linear paths that bend when crossing walls. The key result: wall-crossing automorphisms have positive coefficients, so theta functions expand positively. This also establishes the existence of canonical bases (theta bases) with positive structure constants.

**Generalized cluster algebras (Burcroff-Lee-Mou 2025, PNAS):** Extended positivity to generalized cluster algebras (where exchange polynomials can have higher degree) via **tight gradings** — a new combinatorial object that yields manifestly positive formulas for rank-2 scattering diagrams, then lifts to arbitrary rank. This resolves a decade-old conjecture of Chekhov and Shapiro.

**Quantum positivity (Davison 2018, Annals of Mathematics):** Laurent positivity also holds for quantum cluster algebras, where variables q-commute. Davison's proof uses cohomological Donaldson-Thomas theory and the integrality conjecture for motivic DT invariants.

## Why General Type Was Hard

Mutation sequences grow exponentially with no finite description of the exchange graph in infinite type. Surface-type cluster algebras admit geometric models (triangulations of marked surfaces) that tame this. For general quivers, no such model exists. The GHKK approach sidesteps this by working in a completed algebra where all mutations converge, reducing positivity to counting broken lines in tropical geometry.

## Connection to Representation Theory

Cluster algebras are deeply linked to quiver representations. The **Caldero-Chapoton map** sends indecomposable quiver representations to cluster variables via Euler characteristics of Grassmannians of submodules. For acyclic quivers, the cluster algebra embeds into the coordinate ring of a double Bruhat cell in the corresponding Kac-Moody group (Geiss-Leclerc-Schroer), and cluster monomials sit inside Lusztig's canonical basis. The **cluster category** (Buan-Marsh-Reineke-Reiten-Todorov) categorifies mutation: rigid indecomposables correspond to cluster variables, mutation lifts to tilting in the derived category.

## Computational Tools

SageMath provides `ClusterAlgebra` and `cluster_seed` classes supporting mutation, g-vectors, and F-polynomials. Keller's Java applet computes exchange graphs for finite-type quivers. No general-purpose scattering diagram implementation exists yet, though rank-2 formulas from Burcroff-Lee-Mou are directly computable.

## Relevance to Ergon

Cluster algebras appear wherever discrete dynamics meets algebraic combinatorics. The positivity phenomenon — that combinatorial explosion preserves sign — parallels what we see in tensor structure: operations that should destroy regularity instead preserve it. The GHKK broken-line approach is structurally similar to path-counting in our dissection tensor: both reduce algebraic positivity to tropical/combinatorial enumeration. If the tensor's cross-domain bridges carry a cluster-like exchange structure, positivity of Laurent expansions would explain why bridge coefficients stay nonnegative even as the number of intermediating domains grows.
