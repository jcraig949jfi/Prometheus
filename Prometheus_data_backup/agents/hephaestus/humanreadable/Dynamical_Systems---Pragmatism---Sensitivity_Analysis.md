- negations (“not”, “no”).  

Each proposition is mapped to a basis vector eᵢ in ℝᵏ, forming a sparse binary state vector s₀ ∈ {0,1}ᵏ that marks which primitives appear.  

A dynamical system updates the state via a linear operator A ∈ ℝᵏˣᵏ that encodes logical constraints learned from the prompt:  
- **Transitivity**: if pᵢ→pⱼ and pⱼ→pₖ then A adds weight to pᵢ→pₖ,  
- **Modus ponens**: if pᵢ (antecedent) and pᵢ→pⱼ are present, increase pⱼ,  
- **Consistency**: negations subtract weight from conflicting literals.
