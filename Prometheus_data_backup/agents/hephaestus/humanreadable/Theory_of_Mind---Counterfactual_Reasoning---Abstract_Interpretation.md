## Nous Analysis

**Algorithm**  
We build a lightweight epistemic‑causal reasoner that operates on a set of *possible worlds* represented as boolean vectors.  

1. **Parsing (regex‑based structural extraction)**  
   - Patterns capture:  
     *Negations* (`\bnot\b|\bnever\b`),  
     *Comparatives* (`\bmore than\b|\bless than\b|\bgreater than\b|\bless than or equal\b`),  
     *Conditionals* (`\bif\b|\bunless\b|\bprovided that\b`),  
     *Causal claims* (`\bbecause\b|\bleads to\b|\bresults in\b`),  
     *Ordering* (`\bbefore\b|\bafter\b|\bprecedes\b`),  
     *Mental‑state verbs* (`\bthink\b|\bbelieve\b|\bknow\b|\bintend\b`),  
     *Quantifiers* (`\ball\b|\bsome\b|\bnone\b`).  
   - Each match yields a **Proposition** object: `{id, predicate, polarity, agent?, modality}` where `modality ∈ {actual, belief, counterfactual}` and `agent` is present only for belief modalities.  
   - Conditional statements generate an implication edge `A → B`. Causal claims are treated as deterministic implications.  

2. **Data structures**  
   - `props`: list of Proposition objects (size *n*).  
   - `W`: numpy array of shape `(n, m)` where *m* is the number of worlds; each entry `W[i,j] ∈ {0,1,0.5}` (0 = false, 1 = true, 0.5 = unknown).  
   - `impl`: sparse boolean matrix `(n, n)` for implication edges.  
   - `belief_acc`: dict mapping agent → set of worlds where the agent’s accessibility relation holds (initially all worlds).  

3. **Initialization**  
   - Set `W[i,0] = 1` for facts asserted without negation in the prompt (actual world 0).  
   - Set `W[i,0] = 0` for negated facts.  
   - For each belief statement “Agent X believes P”, copy the truth value of P from world 0 into all worlds in `belief_acc[X]` (over‑approximation: if unknown, keep 0.5).  

4. **Constraint propagation (abstract interpretation + counterfactuals)**  
   - **Modus ponens**: repeat `W_new = np.maximum(W, impl @ W)` until convergence (numpy matrix multiplication treats 0.5 as unknown; any 1 propagates, any 0 blocks).  
   - **Transitivity for ordering**: compute closure of ordering edges similarly.  
   - **Counterfactual worlds**: for each antecedent‑contrary clause “If A had been B”, create a new world *k* by copying `W[:,0]` and flipping the truth of the literal A (and any literals directly constrained by A via `impl`). Initialize `W[:,k]` with this copy, then run the same propagation loop.  
   - The number of worlds *m* grows linearly with the number of distinct counterfactual clauses; each world is an over‑approximation of all concrete worlds reachable under that assumption.  

5. **Scoring a candidate answer**  
   - Parse the answer into a Proposition *q*.  
   - Extract its truth column `t = W[q_id, :]`.  
   - Convert unknowns (0.5) to 0 for a conservative score.  
   - Score = `np.mean(t)` (proportion of worlds where the answer holds).  
   - Optionally weight worlds by inverse Hamming distance from world 0 (fewer flipped antecedents → higher weight) using a pre‑computed weight vector `w` and compute `np.average(t, weights=w)`.
