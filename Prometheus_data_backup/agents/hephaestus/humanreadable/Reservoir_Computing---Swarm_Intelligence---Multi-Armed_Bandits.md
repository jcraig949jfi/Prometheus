## Nous Analysis

The algorithm treats each candidate answer as a particle in a swarm that searches a high‑dimensional feature space generated by a fixed random recurrent reservoir.  

**Data structures**  
- Reservoir: weight matrix **W_res** ∈ ℝⁿˣⁿ (sparse, spectral radius <1) and input matrix **W_in** ∈ ℝⁿˣᵥ (random Gaussian).  
- State vector **xₜ** ∈ ℝⁿ, updated per token.  
- For each candidate *i*: particle position **pᵢ** (the reservoir final state), velocity **vᵢ**, personal best **pᵢᵇᵉˢᵗ**, and personal best score **sᵢᵇᵉˢᵗ**.  
- Swarm global best **gᵇᵉˢᵗ** and its score **sᵍᵇᵉˢᵗ**.  
- Bandit tables: count **cᵢ**, average reward **r̄ᵢ**, and UCB value **uᵢ**.  

**Operations**  
1. **Reservoir encoding** – For each token *w* in the answer, compute one‑hot **e(w)**, then **xₜ = tanh(W_in·e(w) + W_res·xₜ₋₁)**. After the final token, **fᵢ = x_T** is the feature vector.  
2. **Constraint parsing** – Using regex, extract logical atoms: negations (“not”), comparatives (“>”, “<”, “=”), conditionals (“if … then …”), causal cues (“because”, “leads to”), ordering (“before”, “after”), and numeric constraints. Convert each atom to a Boolean variable; build a clause list.  
3. **Constraint propagation** – Initialise truth values from explicit facts; iteratively apply modus ponens and transitivity (matrix multiplication with NumPy) until a fixed point. Count satisfied clauses **sat** and violated clauses **vio**.  
4. **Score** – **scoreᵢ = sat – λ·vio** (λ = 0.5).  
5. **Swarm update** – For each particle:  
   **vᵢ = ω·vᵢ + φ₁·rand()·(pᵢᵇᵉˢᵗ – pᵢ) + φ₂·rand()·(gᵇᵉˢᵗ – pᵢ)**  
   **pᵢ = pᵢ + vᵢ** (position in reservoir space).  
   Re‑encode the perturbed position (by adding a small Gaussian to the reservoir state) to get a new candidate evaluation; update personal/global best if score improves.  
6. **Bandit allocation** – After each evaluation, increment **cᵢ**, update **r̄ᵢ**, compute **uᵢ = r̄ᵢ + √(2·log(Σc)/cᵢ)**. The next candidate to evaluate is the argmax uᵢ, focusing computational budget on promising answers while still exploring.
