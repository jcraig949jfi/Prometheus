   Each proposition is stored as a tuple `(type, args)` in a list; the union of all propositions across prompt \(Q\) and answer \(A_i\) defines a vocabulary \(V\).

2. **Sparse Coding Dictionary** – Build a fixed dictionary \(D\in\mathbb{R}^{|V|\times k}\) (with \(k\ll|V|\)) where each column corresponds to a prototypical logical pattern (e.g., “numeric + comparative”, “negation + conditional”, “causal + ordering”). The dictionary is pre‑computed once using an offline Olshausen‑Field style sparse coding on a corpus of annotated logic‑forms (only numpy needed: iterative soft‑thresholding).  
   For each text we solve \(\min_{\alpha}\|x-D\alpha\|_2^2+\lambda\|\alpha\|_1\) (ISTA) to obtain a sparse code \(\alpha\in\mathbb{R}^k\). The code is the representation of the text’s logical structure.

3. **Measure‑Theoretic Weighting** – Define a σ‑algebra \(\mathcal{F}\) on the space of codes generated by all possible sparse vectors (discretise each coefficient to a grid of step δ). Assign a Lebesgue‑like measure \(\mu\) that favours codes with low ℓ₁‑norm (simplicity) via density \(d\mu(\alpha)\propto e^{-\beta\|\alpha\|_1}d\alpha\). The normalising constant is computed analytically (product of 1‑D Laplace integrals).

4. **Kolmogorov‑Complexity Approximation** – Approximate the description length of a candidate answer by the **negative log‑measure** of its code:  
   \[
   L(A_i|Q) = -\log \mu(\alpha_{A_i|Q}) + C,
   \]
   where \(\alpha_{A_i|Q}\) is the sparse code of the answer after projecting out the prompt’s code (i.e., \(\alpha_{A_i|Q}=\alpha_{A_i}-\text{proj}_{\alpha_Q}\alpha_{A_i}\)), and \(C\) is a universal constant absorbed into λ. This is a computable proxy for Kolmogorov complexity because shorter codes receive higher measure.

5. **Scoring** – The final score for answer \(A_i\) is  
   \[
   S(A_i)= -L(A_i|Q) = \log \mu(\alpha_{A_i|Q}),
   \]
   i.e., higher scores for answers that are both logically aligned with the prompt (small residual code) and descriptively simple (high measure). Scores are normalised to \([0,1]\) for comparison.
